# Job Application Tracking & Insight System

A Python-based backend system that tracks job applications and generates data-driven insights to improve job search strategy.

## Features
- Store and manage job applications
- Track application status changes
- Calculate response time metrics
- Analyze effectiveness by application source
- Identify stale applications requiring follow-up

## Tech Stack
- Python
- SQLite
- SQL
- datetime module

## Why This Project
Built to turn an unstructured job search process into a data-driven system after applying to 500+ jobs.

## Future Improvements
- Web interface (Streamlit / FastAPI)
- Email follow-up reminders
- Export insights to reports
