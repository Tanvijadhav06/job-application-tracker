# Job Application Tracking & Insight System (AI-Enhanced)

A Python-based backend system that tracks job applications and generates data-driven and AI-powered insights to improve job search strategy.

Features

- Store and manage job applications
- Track application status changes
- Calculate response time metrics
- Analyze effectiveness by application source
- Identify stale applications requiring follow-up
- Generate AI-based insights from application data using NLP models

Tech Stack

- Python
- SQLite
- SQL
- datetime module
- Hugging Face Transformers (NLP)

AI Enhancement

The system integrates a transformer-based NLP model to analyze application data and generate high-level insights such as:

- Trends across roles and companies  
- Response patterns and delays  
- Overall application performance  

This combines rule-based analytics with AI-generated summaries for better decision-making.

#Why This Project

Built to convert an unstructured job search process into a structured, data-driven system after applying to 500+ jobs, and further enhanced with AI to extract meaningful insights automatically.
